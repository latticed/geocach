# Rätsel-Vordrucke — zum Ausdrucken & Ausfüllen

Leere Vorlagen für eigene Rätsel (Outdoor-Hinweise & Indoor-Stationen).
Pro Rätsel ein Vordruck ausfüllen, ausschneiden, an der Station/im Cache platzieren.
Lösungs-Ziffer nicht vergessen, falls ihr den Schluss-Code nutzt (siehe `Regenplan-Stationen.md`).

---

## 2) Schluss-Code-Sammelblatt (für die Suchgruppe)

> Ein Blatt pro Gruppe — Ziffern unterwegs eintragen, am Ende ergibt sich der Code.

```
SCHATZ-CODE SAMMELN                Gruppe: A / B

Station 1: Ziffer ___    Station 4: Ziffer ___
Station 2: Ziffer ___    Station 5: Ziffer ___
Station 3: Ziffer ___    Station 6: Ziffer ___

►  SCHLUSS-CODE (in Reihenfolge): ___ ___ ___ ___ ___ ___
```

## 3) Rätsel-Ideen-Baukasten (zum Abschreiben/Anpassen)

Damit ihr die Vordrucke schnell füllen könnt — Lösungen in Klammern:

- **Anagramm:** Buchstaben einer Sache mischen → „HCSTAZ" (= SCHATZ)
- **Zahlenreihe:** 2, 4, 8, 16, __ (= 32)
- **Buchstaben = Zahlen** (A=1, B=2 …): „6-12-21-18" (= FLUR)
- **Caesar-Code** (jeder Buchstabe +1): „TDIBU{" → bewusst knifflig (= SCHATZ-Variante)
- **Rätselfrage:** „Ich habe Blätter, bin aber kein Baum." (= Buch)
- **Quersumme/Mathe:** „Tisch-Nr. = Quersumme von 1+0+1" (= Tisch 2)
- **Spiegelschrift:** Hinweis spiegelverkehrt drucken
- **Such-/Bewegungsaufgabe:** „10 Hampelmänner, dann Umschlag öffnen"
- **Bilderrätsel:** kleines Symbol = Ort (z. B. Buch = Bibliothek)

## 4) QR-Hinweis-Etikett (Blanko)

> Für Indoor-QR-Rallye: QR aufkleben, Klartext-Backup daneben (falls Handy streikt).

```
┌──────────────────────────┐   ┌──────────────────────────┐
│   [ QR-CODE HIER         │   │ Backup-Hinweis (Text):    │
│     AUFKLEBEN ]          │   │ _______________________   │
│                          │   │ _______________________   │
│  Station: ____           │   │ Lösung-Ziffer: ____       │
└──────────────────────────┘   └──────────────────────────┘
```

> **QR erstellen:** Hinweistext in einen QR-Generator (Typ „Text") einfügen, ausdrucken, aufkleben.
> Reiner Text-QR funktioniert **offline**.
